# Ressources pour l'agregation de SII, option IM

L'objectif de ce site est de regrouper diverse ressources pouvant servir de support ou d'inspiration pour la préparation de séquence pédagogique

Cela inclu entre autres:
- Lien vers des sites institutionnels comme eduscol
- Lien vers des ressources publiées par certains lycées
- Exemples d'emploi du temps de divers filière
- Template draw.io pour certains diagrammes commun (digramme des exigences, chaine d'énergie/information...)
